## 1 linux -- less

**less is more.**

`less [-options] <file>`

>`[]`表示可选，`<>`表示必选

options|desc
:-|:-
-b |《缓冲区大小》 设置缓冲区的大小
-e |当文件显示结束后，自动离开
-f |强迫打开特殊文件，例如外围设备代号、目录和二进制文件
-g |只标志最后搜索的关键词
-i |忽略搜索时的大小写
-m |显示类似more命令的百分比
-N |显示每行的行号
-o |《文件名》 将less 输出的内容在指定文件中保存起来
-Q |不使用警告音
-s |显示连续空行为一行
-S |行过长时间将超出部分舍弃
-x |《数字》 将“tab”键显示为规定的数字空格

常用命令：`less -mN <file>`
